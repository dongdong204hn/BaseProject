– dateForKey:

若key对应值为NString, 根据string长度返回形式为"yyyy-MM-dd HH:mm"或"yyyy-MM-dd HH:mm:ss"的NSDate, 若string无法解析则返回nil; 若key对应值为数字, 则返回以该数字为时间戳对应的NSDate; 其余情况返回nil.

- (NSDate *)dateForKey:(NSString *)key

Return Value

若key对应值为NString, 根据string长度返回形式为"yyyy-MM-dd HH:mm"或"yyyy-MM-dd HH:mm:ss"的NSDate, 若string无法解析则返回nil; 若key对应值为数字, 则返回以该数字为时间戳对应的NSDate; 其余情况返回nil.
